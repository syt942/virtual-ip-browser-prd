#### Build Failure Response

```markdown
IF Linux build fails:
  1. Check logs: Actions → build-linux → View logs
  2. Common issues: Missing rpm dependency
  3. Fix: Update workflow or rebuild

IF Windows build fails:
  1. Check code signing: CSC_LINK secret valid?
  2. Check NSIS: npm run package:win logs
  3. Alternative: Build without signing (mark in release)

IF macOS build fails:
  1. Check signing: Apple certificates valid?
  2. Check notarization: APPLE_ID credentials
  3. Alternative: Skip notarization for beta
```
