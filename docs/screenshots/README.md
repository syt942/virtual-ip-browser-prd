# Screenshot Placeholders

This directory contains placeholder images for the User Guide documentation.

## Required Screenshots

- main-interface.png - Main browser interface
- tab-bar.png - Tab bar with multiple tabs
- add-proxy-dialog.png - Add proxy dialog
- rotation-settings.png - Proxy rotation configuration
- privacy-panel.png - Privacy protection settings
- tab-management.png - Tab management interface
- automation-panel.png - Automation configuration
- domain-targeting.png - Domain targeting settings
- creator-support.png - Creator support interface
- settings-panel.png - Application settings

## Creating Screenshots

1. Run the application: `npm run dev`
2. Navigate to each feature
3. Capture screenshots at 1280x720 or higher
4. Save as PNG with the names above
